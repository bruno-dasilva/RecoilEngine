
Recoil engine is an RTS game engine that supports large-scale RTS games, designed to handle thousands of units at once.

The performance target for the engine is 10k units concurrently, but that includes buildings. Mobiles units tend to be about maybe 40% of the total unit count in late games. The largest number of units in one game so far is ~17.7k (but that's not something we're targeting).
